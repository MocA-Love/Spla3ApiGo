# What's up

指定されたマッチタイプのスケジュールをAPIから取得し、独断と偏見で見やすいと思う形式で出力

## Requirement
Go 1.22.4

## Usage

1. Clone:

    ```bash
    git clone https://github.com/MocA-Love/Spla3ApiGo.git
    cd Spla3ApiGo
    ```

2. Ruuuuuuuun:

    ```bash
    go run .
    ```
